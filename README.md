# Restaurant Reservation API (Nest.js + GraphQL)

## Requirements Implemented
✔ GraphQL (Queries + Mutations)  
✔ Mongoose schemas (mocked storage)  
✔ Custom Date scalar  
✔ Reservation time validation  
✔ reservationCount field resolver  
✔ Strict TypeScript

## Run the project

npm install
npm run start

## For tests

npm run test
